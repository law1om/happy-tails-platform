# Happy Tails 🐾

Приложение для усыновления животных из приюта.

## Запуск

Автоматически создастся:
- Таблицы в БД
- Пользователи (admin@happytails.com / admin123, user@test.com / password123)
- 12 животных (6 собак + 6 кошек)
- 5 событий

### Frontend (Flutter)

```bash
flutter pub get
flutter run -d chrome
```

## Функции

### Пользователь
- Просмотр каталога животных
- Подача заявок на усыновление
- Создание анкеты усыновителя
- Просмотр событий

### Администратор
- Управление животными (добавление фото через URL)
- Управление заявками
- Управление событиями
- Управление пользователями

## Технологии

- **Backend**: Spring Boot, PostgreSQL, JWT
- **Frontend**: Flutter, Dart
- **API**: REST
